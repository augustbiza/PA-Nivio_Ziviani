# Quicksort - Análise do algoritmo
### Comparações
#### Melhor: (n*Log(n))
#### Pior: O(n^2)
#### Médio: (n*Log(n))

### Movimentações
#### Melhor: (n*Log(n))
#### Pior: O(n^2) 
#### Médio: (n*Log(n))

### Estabilidade
#### Não estável

### Vantagens
#### Complexidade MÉDIA linear-algoritmica: O(n*Log(n))

### Desvantagens
