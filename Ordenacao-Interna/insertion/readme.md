# Insertion sort - Análise do algoritmo
### Comparações
#### Melhor: O(n)
#### Pior: O(n^2)
#### Médio: O(n^2)

### Movimentações
#### Melhor: O(n)
#### Pior: O(n^2)
#### Médio: O(n^2)

### Estabilidade
#### Estável

### Vantagens
#### Complexidade linear em arquivos quase ordenados

### Desvantagens
#### Complexidade quadrática em arquivos todo desordenados

### Bom
#### Arquivos quase ordenados
### Ruim
#### Arquivos todo desordenado
